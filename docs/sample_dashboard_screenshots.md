# Sample Dashboard Screenshots

This folder is reserved for dashboard screenshots used in GitHub documentation and portfolio walkthroughs.

Suggested captures:

1. Executive KPI overview
2. Workforce participation and placement trend analysis
3. Program ranking and workforce readiness view
4. Funding allocation and efficiency analysis
5. County and logistics specialization summaries

Run the application locally with:

```bash
streamlit run app.py
```

Then capture screenshots from the browser for inclusion in the README.
